## Product hierarchy

- Companion is the default product surface. Model and tool controls are secondary.
- Persona owns character cards, relationship framing, voice, appearance, and advanced bindings.
- Memory defaults to a readable relationship archive; the graph remains an advanced inspection view.
- Capabilities contains reminders, workflows, packages, Skill, MCP, Shizuku, terminal, and automation.
- Me contains global model, data, privacy, appearance, and developer settings.
