# Mira 气泡装扮

## 原状

聊天主题支持气泡颜色、文字颜色、自定义图片、玻璃效果、头像和左右内边距，但圆角半径与气泡宽度写死。用户需要逐项组合参数，也缺少能直接套用的协调主题。

## 目标

- 提供 Mira 蓝、微信绿、樱花粉、薄荷绿、柔雾紫、极简黑白六套气泡主题
- 套装一次协调气泡色、文字色、头像、圆角、宽度、间距和阴影
- 在气泡装扮首页直接提供双方气泡色和文字色入口，不要求用户进入深层高级设置
- 保留图片气泡、玻璃气泡、字体与头像等已有高级能力
- 支持用户与 Mira 独立调节圆角和最大宽度
- 支持调节消息留白与气泡阴影
- 新参数继续参与角色卡和角色群组主题复制

## 作用域

- `UserPreferencesManager` 与 `ThemePreferenceSnapshot`
- 主题设置聊天页及实时预览
- 用户与 AI 气泡 Compose 渲染
- Web Chat 主题数据模型
- 中英文设置文案

## 验收

- 点击套装后聊天样式切换为气泡，并关闭冲突的图片与玻璃背景
- 手动调整参数后显示为自定义搭配
- 普通气泡、图片气泡与玻璃气泡使用统一圆角、宽度和间距参数
- 阴影参数作用于纯色与图片气泡，玻璃气泡保留专用光学阴影
- 角色卡主题保存、切换和复制包含新增参数
